Deklarasi Variabel (Membuat Variabel)

Untuk membuat variabel di JavaScript, kita menggunakan kata kunci khusus. Ada 3 cara utama:

  1. var (Cara Lama)
    2. var namaPembeli;
      var harga = 50000;
    • Fungsi: Deklarasi variabel konstan yang tidak bisa diubah.
    • Bisa diubah: Tidak, nilai harus langsung diassign dan tidak bisa berubah.
    • Rekomendasi: Gunakan untuk nilai yang tetap sepanjang kode.
  2. let (Cara Modern - Dapat Diubah)
    3. let usia = 25;
    let alamat;
    usia = 26; // Nilai bisa diubah
    • Fungsi: Deklarasi variabel dengan scope blok {}.
    • Bisa diubah: Ya, nilainya bisa diubah setelah dideklarasikan.
    • Rekomendasi: Gunakan untuk nilai yang perlu diubah nanti.
  3. const (Cara Modern - Konstan/Tetap)
    4. const pi = 3.14;
    const namaAplikasi = "MyApp";
    // pi = 3.15; // ERROR! Tidak bisa diubah
    • Fungsi: Deklarasi variabel konstan yang tidak bisa diubah.
    • Bisa diubah: Tidak, nilai harus langsung diassign dan tidak bisa berubah.
    • Rekomendasi: Gunakan untuk nilai yang tetap sepanjang kode.
