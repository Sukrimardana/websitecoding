Aturan Penamaan Variabel

  1. Boleh: huruf, angka, $, _
  2. Tidak boleh: diawali angka, spasi, karakter khusus
  3. Case sensitive: nama ≠ Nama ≠ NAMA
  4. Tidak boleh: menggunakan kata kunci JavaScript (seperti let, if, function)
// Benar
  let namaLengkap;
  let _privateData;
  let $element;
  let user2;

// Salah
  let 2ndPlace; // diawali angka
  let full name; // ada spasi
  let let; // kata kunci
contoh pengunaan:
// Deklarasi variabel
  let nama = "Budi Santoso";
  const umur = 30;
  let isActive = true;

// Menggunakan variabel
  console.log("Nama: " + nama); // Output: Nama: Budi Santoso
  console.log("Umur: " + umur + " tahun"); // Output: Umur: 30 tahun

// Mengubah nilai variabel let
  nama = "Budi Raharjo";
  isActive = false;

  console.log("Nama baru: " + nama); // Output: Nama baru: Budi Raharjo
Tipe Data yang Disimpan

Variabel JavaScript dapat menyimpan berbagai tipe data:

// String (teks)
  let greeting = "Hello World!";

// Number (angka)
  let count = 42;
  let price = 19.99;

// Boolean (true/false)
  let isLoggedIn = true;
  let hasPermission = false;

// Array (kumpulan nilai)
  let fruits = ["Apple", "Banana", "Orange"];

// Object (data terstruktur)
  let person = {
    name: "Alice",
    age: 25,
    city: "Jakarta"
  };

// Null (kosong)
  let emptyValue = null;

// Undefined (belum didefinisikan)
  let notDefined;
Kesimpulan

Aspek Penjelasan Fungsi Menyimpan dan mengelola data dalam program Cara Modern Gunakan let untuk nilai yang berubah, const untuk nilai tetap Keuntungan Code lebih mudah dibaca, diubah, dan dikelola Best Practice Gunakan nama yang deskriptif dan konsisten Variabel adalah fondasi dasar pemrograman yang memungkinkan Anda menyimpan, mengakses, dan memanipulasi data dalam program JavaScript Anda.
