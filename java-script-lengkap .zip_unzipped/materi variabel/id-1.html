Apa Itu Variabel

Variabel dalam JavaScript adalah sebuah wadah atau penyimpanan bernama yang digunakan untuk menampung data (nilai) di dalam memori komputer. Bayangkan variabel seperti sebuah kotak yang memiliki nama label dan Anda bisa memasukkan berbagai barang (data) ke dalamnya.

Analogi Sederhana

  • Nama Variabel = Label pada kotak (misal: "kotakMainan")
  • Variabel = Kotak itu sendiri
  • Nilai = Isi kotak (misal: bola, boneka, mobil-mobilan)
