Assignment Operator di JavaScript

Assignment operator digunakan untuk memberikan nilai ke variabel. Berikut adalah penjelasan lengkap tentang berbagai jenis assignment operator di JavaScript:
  1. Assignment Operator Dasar (=)
    2. let x = 10; // Memberikan nilai 10 ke variabel x
    let nama = "John"; // Memberikan string "John" ke variabel nama
    let isActive = true; // Memberikan nilai boolean true
  2. Compound Assignment Operators

    3. Operator yang menggabungkan operasi aritmatika dengan assignment:


      Addition Assignment (+=)
    let a = 5;
    a += 3; // Sama dengan: a = a + 3
    console.log(a); // Output: 8

    let text = "Hello";
    text += " World"; // Sama dengan: text = text + " World"
    console.log(text); // Output: "Hello World"
      Subtraction Assignment (-=)
    let b = 10;
    b -= 4; // Sama dengan: b = b - 4
    console.log(b); // Output: 6
      Multiplication Assignment (*=)
    let c = 7;
    c *= 2; // Sama dengan: c = c * 2
    console.log(c); // Output: 14
      Division Assignment (/=)
    let d = 20;
    d /= 4; // Sama dengan: d = d / 4
    console.log(d); // Output: 5
      Remainder Assignment (%=)
    let e = 15;
    e %= 4; // Sama dengan: e = e % 4
    console.log(e); // Output: 3 (karena 15 รท 4 = 3 sisa 3)
      Exponentiation Assignment (**=)
    let f = 2;
    f **= 3; // Sama dengan: f = f ** 3 (2 pangkat 3)
    console.log(f); // Output: 8
  3. Bitwise Assignment Operators
      4. Left Shift Assignment (<<=)< /em>
    let g = 5; // Binary: 101
    g <<= 1; // Sama dengan: g=g << 1 (Binary: 1010)
    console.log(g); // Output: 10
      Right Shift Assignment (>>=)
    let h = 10; // Binary: 1010
    h >>= 1; // Sama dengan: h = h >> 1 (Binary: 101)
    console.log(h); // Output: 5
      Unsigned Right Shift Assignment (>>>=)
    let i = -10;
    i >>>= 1; // Shift right tanpa memperhatikan tanda
    console.log(i); // Output: 2147483643 (pada 32-bit)
      Bitwise AND Assignment (&=)
    let j = 5; // Binary: 101
    j &= 3; // Sama dengan: j = j & 3 (Binary: 001)
    console.log(j); // Output: 1
      Bitwise OR Assignment (|=)
    let k = 5; // Binary: 101
    k |= 3; // Sama dengan: k = k | 3 (Binary: 111)
    console.log(k); // Output: 7
      Bitwise XOR Assignment (^=)
    let l = 5; // Binary: 101
    l ^= 3; // Sama dengan: l = l ^ 3 (Binary: 110)
    console.log(l); // Output: 6
  4. Logical Assignment Operators (ES2021)
      5. Logical AND Assignment (&&=)
    let user = { name: "John" };
    user.name &&= "Anonymous"; // Hanya assign jika user.name truthy
    console.log(user.name); // Output: "John"

    let user2 = { name: "" };
    user2.name &&= "Anonymous"; // Tidak assign karena user2.name falsy
    console.log(user2.name); // Output: ""
      Logical OR Assignment (||=)
    let config = { theme: null };
    config.theme ||= "dark"; // Assign "dark" jika config.theme falsy
    console.log(config.theme); // Output: "dark"

    let config2 = { theme: "light" };
    config2.theme ||= "dark"; // Tidak assign karena config2.theme truthy
    console.log(config2.theme); // Output: "light"
      Nullish Coalescing Assignment (??=)
    let settings = { volume: 0 };
    settings.volume ??= 50; // Assign 50 hanya jika settings.volume null atau undefined
    console.log(settings.volume); // Output: 0 (karena 0 bukan null/undefined)

    let settings2 = { volume: null };
    settings2.volume ??= 50; // Assign 50 karena settings2.volume null
    console.log(settings2.volume); // Output: 50
  5. Chaining Assignment
    6. let x, y, z;
    x = y = z = 10; // Semua variabel mendapat nilai 10
    console.log(x, y, z); // Output: 10 10 10
  6. Destructuring Assignment
    7. // Array Destructuring
    const [first, second] = [1, 2];
    console.log(first, second); // Output: 1 2

    // Object Destructuring
    const { name, age } = { name: "Alice", age: 25 };
    console.log(name, age); // Output: "Alice" 25

    // Dengan default values
    const [a = 1, b = 2] = [10];
    console.log(a, b); // Output: 10 2

    // Swapping variables
    let m = 1, n = 2;
    [m, n] = [n, m];
    console.log(m, n); // Output: 2 1
      Contoh praktis
    // Perhitungan total belanja
    let total = 0;
    let hargaBarang = 10000;
    let jumlah = 3;

    total += hargaBarang * jumlah; // total = 30000
    total *= 0.9; // Diskon 10%, total = 27000
    console.log(`Total belanja: Rp${total}`);

    // Update nilai berdasarkan kondisi
    let score = 85;
    score += score > 80 ? 10 : 5; // Tambah 10 jika score > 80
    console.log(score); // Output: 95

    Tips Penting!

  1. Gunakan compound assignment untuk kode yang lebih ringkas
  2. Perhatikan urutan operasi (operator precedence)
  3. Logical assignment operators sangat berguna untuk nilai default
  4. Destructuring assignment sangat powerful untuk bekerja dengan array dan object
