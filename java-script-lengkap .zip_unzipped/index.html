Apa Itu Java Script?

Pengertian Java Script

JavaScript adalah bahasa pemrograman tingkat tinggi yang digunakan terutama untuk membuat halaman web yang interaktif dan dinamis. Berbeda dengan HTML (yang mengatur struktur) dan CSS (yang mengatur tampilan), JavaScript memberikan "kehidupan" pada halaman web dengan menambahkan perilaku dan fungsionalitas.

Sejarah Singkat


  • Diciptakan tahun 1995 oleh Brendan Eich di Netscape
  • Awalnya disebut "Mocha", kemudian "LiveScript", akhirnya "JavaScript"
  • Tidak ada hubungan dengan Java - namanya hanya strategi marketing saat itu
  • Sekarang menjadi salah satu bahasa pemrograman paling populer di dunia

Karakteristik Utama


  • Bahasa scripting - tidak perlu dikompilasi, dijalankan langsung oleh browser
  • Multi-paradigma - mendukung pemrograman prosedural, berorientasi objek, dan fungsional
  • Dinamis - tipe data ditentukan otomatis saat runtime
  • Interpreter-based - dijalankan baris per baris

Fungsi dan Kegunaan JavaScript

  1. Interaktivitas Website
    • Menanggapi klik mouse, input keyboard, dll.
    • Validasi form sebelum dikirim ke server
    • Animasi dan efek visual
  2. Manipulasi DOM
    • Mengubah konten, struktur, dan style halaman web secara dinamis
    // Contoh mengubah teks elemen
        document.getElementById("demo").innerHTML = "Halo JavaScript!";
  3. Web Development
    • Frontend (React, Vue, Angular)
    • Backend (Node.js)
    • Mobile apps (React Native)
    • Desktop apps (Electron)>
  4. Async Programming
    • Fetch data dari server tanpa reload halaman (AJAX)
    • Berkomunikasi dengan API
      Contoh Kode ederhana

Ekosistem Modern

  • Framework/Library: React, Vue, Angular, Svelte
  • Runtime: Node.js (untuk server-side)
  • Tools: npm, yarn, webpack, babel
  • APIs: DOM API, Fetch API, Geolocation API, dll.
